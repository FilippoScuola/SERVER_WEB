// ABOUT — PYTHON TCP RAW SERVER

IL PROGETTO

// OBIETTIVO

Implementazione di un web server HTTP da zero in Python puro, senza framework esterni. Il server accetta connessioni TCP, legge le richieste HTTP e restituisce i file statici configurati.

// ARCHITETTURA

Il server legge la propria configurazione da un file YAML esterno: host, porta, route, MIME types e pagine di errore. Ogni connessione viene gestita in un thread separato con un semaforo per limitare la concorrenza massima.
